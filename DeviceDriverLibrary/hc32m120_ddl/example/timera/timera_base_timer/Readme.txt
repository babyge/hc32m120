功能描述
================================================================================
本样例展示TimerA模块基本计数功能。

说明：
本样例设置TimerA为锯齿波、向上计数模式，TimerA单元1开启计数上溢中断，配置计数周
期为10ms，TimerA单元1每秒切换LED_R状态。

使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载程序功能，关闭IDE下载界面；
3）观察LED_R闪烁状态，连续闪烁则工作正常。
