功能描述
================================================================================
本样例展示TimerB模块级联为32位计数器功能。

说明：
本样例设置TimerB为锯齿波、向上计数模式，TimerB单元1周期计数时间为100ms，单元2周期
间隔计数为10，使能单元2计数上溢中断，单元2上溢中断翻转RGB绿色灯引脚，RGB绿色灯闪烁
，闪烁周期为2S。

使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载程序功能，关闭IDE下载界面；
3）观察RGB绿色灯状态，不停闪烁，闪烁周期为2S。
