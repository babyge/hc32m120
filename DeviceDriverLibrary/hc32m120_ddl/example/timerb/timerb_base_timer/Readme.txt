功能描述
================================================================================
本样例展示TimerB模块基本计数及同步启动功能。

说明：
本样例设置TimerB为锯齿波、向上计数模式，TimerB单元1同步启动单元2，单元1、2周期计
数值相同，使能计数上溢中断，单元1、2上溢中断分别间隔一次溢出计数后，翻转RGB绿色
灯引脚，RGB绿色灯闪烁。

使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载程序功能，关闭IDE下载界面；
3）观察RGB绿色灯状态，不停闪烁，闪烁频率为1Hz。
