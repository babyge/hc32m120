功能描述
================================================================================
本例程展示了TIMER2硬件控制器的基本功能，实现了用事件EVT_PORT_EIRQ2启动TIMER2，用
TRIGA的下降沿停止TIMER2。

================================================================================
测试环境
================================================================================
测试用板:
---------------------
STK_HC32M120_LQFP48_050_V11

辅助工具:
---------------------
无

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译，下载至目标板；
2）按下按键SW1，红色LED以10Hz频率闪烁；
3）在P30制造一个下降沿，红色LED停止闪烁。
