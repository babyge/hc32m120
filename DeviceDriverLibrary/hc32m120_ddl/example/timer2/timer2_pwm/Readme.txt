功能描述
================================================================================
本例程展示了TIMER2的PWM输出功能。

================================================================================
测试环境
================================================================================
测试用板:
---------------------
STK_HC32M120_LQFP48_050_V11

辅助工具:
---------------------
无

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译，下载至目标板；
2）将示波器探头连接至P30，可看到示波器显示频率为2KHz、占空比为50%的PWM。

================================================================================
注意
================================================================================
TIMER2的PWM占空比恒为50%，不可调。
