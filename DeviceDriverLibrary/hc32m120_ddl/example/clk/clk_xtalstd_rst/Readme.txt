功能描述
================================================================================
本样例为Xtal故障诊断，检测到故障产生复位，点亮红灯。

================================================================================
测试环境
================================================================================
测试用板:
---------------------
STK_HC32M120_LQFP48_050_V11

辅助工具:
---------------------
镊子

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的下载和调试功能，全速运行；
3）发现LED1（绿）亮；
4）用镊子短接外部晶振（制造XTAL故障）；
5）LED1（绿）灭，LED0（红）亮。
