功能描述
================================================================================
本样例主要展示如何使用Timer4PWM死区定时器滤波模式。

说明：
样例配置比较输出模块OCO低通道，并通过PWM死区定时器滤波模式，输出其波形。

================================================================================
测试环境
================================================================================
测试用板:
---------------------
STK_HC32M120_LQFP48_050_V11

辅助工具:
---------------------
示波器Tektronix DPO2024B

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）测试板J3-P61(TIM4_1_OUH)、J3-P60(TIM4_1_OUL)引脚与示波器相连；
2）编译、下载并运行；
3）观察示波器，J3-P61、J3-P60通道波形周期相同，占空比为1/2，极性相反，周期为1S
   为CNT周期2倍。
