功能描述
================================================================================
本样例主要展示如何使用Timer4 PWM重载定时器功能。

说明：
PWM设置为直通模式和死区定时器模式时，PFSR可用于16位重载计数器功能
样例配置PWM为死区定时器模式，设置PFSR，使其产生定时器中断

================================================================================
测试环境
================================================================================
测试用板:
---------------------
STK_HC32M120_LQFP48_050_V11

辅助工具:
---------------------
示波器Tektronix DPO2024B

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）测试板J3-P63引脚与示波器相连；
2）编译、下载并运行；
3）观察示波器，J3-P63输出周期脉冲，脉冲频率为1Hz，与重载定时器周期一致。
