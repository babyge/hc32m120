功能描述
================================================================================
本例程实现了SPI从机模式的配置和用法。


================================================================================
测试环境
================================================================================
测试用板:
---------------------
STK_HC32M120_LQFP48_050_V11

辅助工具:
---------------------
无

辅助软件:
---------------------
无

================================================================================
使用步骤
================================================================================
1）打开工程并重新编译；
2）启动IDE的调试功能，断点查看程序执行情况；


================================================================================
注意
================================================================================
1、从机模式下，SCK最高频率为PCLK/6；
2、本例程SPI工作模式为全双工模式，使能接收中断。
